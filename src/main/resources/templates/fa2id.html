Distributed Room Reservation System


Sep 2017 – Jan 2018

This is an academic project at Concordia University for the course COMP6231 Distributed System Design.

The main goal of the project was to implements a synchronized multithreaded system with three servers each representing a campus. There are 2 different clients in this program who can connect and use the system such as students and administrators.

Handling crash and software failures using replicas and replica managers.

Administrators can make and delete rooms for every campus.
Students can book a room, cancel or change their bookings.

Servers are connected together using UDP (reliable UDP), and clients are connected to the servers using JavaRMI and CORBA.

Technologies used in this project:

- Java
- JavaRMI
- CORBA
- UDP
- RUDP
- GSON

Knights vs Aliens


Jan 2017 – Apr 2017

This is an academic project (3D Game) for the course Advanced Game Development COMP6331 at Concordia University.

The game is made by Unity game engine, and we have used C# as our preferred programming language for scripting and implementing various algorithms.

Genre: Tower Defense

Setting: Medieval fantastic with alien invasion

Summary: The players will build towers in order to help him defend against a horde of alien from destroying the castle. The can player creates a maze to lengthen the path the alien will have to take to reach the castle while waves of aliens will keep pushing forward to reach their goal.

Mechanics: Resource collection - Creating and upgrading Towers using resources to defend the castle - Building a maze to maximize the efficiency of the towers - Build phase to plane the maze - defense phase where waves of enemies tries to reach the castle

Movement AI: Steering Behaviour(wall avoidance, arrive, seek, flee, wander)
Path-Finding: A* pathfinding through the grid based level and partial pathfinding
Decision-Making: Finite State Machine
Other features: Fuzzy Logic

*** This game ranked as the second game between 15 games created by 15 groups of students with game programming background
